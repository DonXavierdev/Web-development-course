#### basics of JS.
```
JavaScript is a programming language that allows developers to manipulate data in many ways, including working with strings. In JavaScript, a string is a sequence of characters enclosed in quotes, either single quotes ('') or double quotes ("").
The <script> tag is used to embed client-side script (JavaScript) into HTML pages.
The <script> element either contains the script itself, or it points to an external script file using the src property.
Here are some of the most commonly used string functions in JavaScript:

length: This function returns the length of the string in characters.

console.log(str.length)
indexOf: This function returns the index of the first occurrence of a specified value in a string.
console.log(str.indexOf('l'))
substring: This function returns a part of a string between two specified indices.

console.log(str.substring(1, 4))
replace: This function replaces a specified value with another value in a string.

console.log(str.replace('world', 'JavaScript'))

toUpperCase: This function converts a string to uppercase.

console.log(str.toUpperCase())
toLowerCase: This function converts a string to lowercase.
console.log(str.toLowerCase());
split : split() function is used to split a string into an array of substrings based on a specified separator.
const arr = str.split(",")
```
