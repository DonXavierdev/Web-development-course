# NOTES
#### HTML basics
```
some major html tags include:
<html> : every html doc begins with this tag, all other elements are descendents of this tag.
<h1> to <h6> : Html provides different header tags ranging from h1 to h6
<body> : the content of a web page is wrapped in <body> tag
<p> : Normal text can be included in a web page using <p> element.
<table> : We can use the <table> element to display data in tabular format.
<form> : used to create an HTML form that allows users to input data and send it to a server.
```
#### CSS basics
```
To include css into our webpage we use:
Inline : <h1 style="color: red;">Heading</h1>
Embedded : using <style> ... </style> inside the html file
External : <link rel="stylesheet" href="path/to/my/custom.css">
CSS contains selectors to target specific HTML elements on a page which include:
```
- element selector
- class selector 
- id selector
- attribute selector
